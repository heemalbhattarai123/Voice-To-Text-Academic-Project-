Some Important Notes before Implementing the codes.
1. You must have XAMPP Server installed on your machine.
2. Please save GadaTechnology file inside htdocs. htdocs is found inside Xampp which is inside C drive.
3. Open Mysql admin on Chrome (recommended browser) and export the given sql database into it.
4. Now open new tab on chrome and type http://localhost/GadaTechnology/welcome.html
5. Admin Account is already created. Admin just need to enter admin for both user name and password to access admin page.
6. New User need to be created by signing up from the link given on http://localhost/GadaTechnology/welcome.html page.



Plese refer to User Manual for more details.
